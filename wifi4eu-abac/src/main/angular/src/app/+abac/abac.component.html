Legal Entities (LEF)


Check Statuts of LEFs in ABAC

This button will check in ABAC the status of the LEFs
which request creation. In case there is a regected LEF,
it will display a screen (country + municility + ABAC ref)
so you can provide the correct ABAC reference number
for all regected LEFs. It will create a file which
you can download a later must import into the
Wifi4eu Portal.



Budgetary Commitment (BC)


Check Statuts of BCs in ABAC

Once create by the Wifi4eu Interface and validated by your
AOSD, clicking on this button will allow you to retrieve the data
in a file so to updated the portal.
